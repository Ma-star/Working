\subsection{问题一：机器断电挂了，重启服务，计算节点的ovs无法启动}
\begin{itemize}
	\item[1.]断电之后，再重启机器导致时间不一致。计算节点显示时间是2013年日期，其他节点正常
	\item[2.]虚拟机一直无法创建成功。
	\item[3.]由于系统时钟和原子时钟时间偏差较大，使用ntp不起作用，其他的设置时间也不起作用
	\item[4.]需手动设置，使用如下命令
		\begin{lstlisting}
			ntpdate time.nist.gov
		\end{lstlisting}
	\item[5.]中国国家授时中心：210.72.145.39。美国国家授时中心：time.nist.gov
	\item[6.]详情见链接：http://www.pubyun.com/blog/deveops/linux下的时钟设置和时区
\end{itemize}
