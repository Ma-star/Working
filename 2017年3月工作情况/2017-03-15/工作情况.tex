\subsection{任务二：neutron的l3-agent的三种高可用方案}
\begin{itemize}
	\item[1.]automatic-l3-agent-failover:需要多个网络节点，主要问题是切换时间太长
	\item[2.]vrrp:通过keepalived软件实现，需要多余的一个网络节点实现A/P，成本较高和扩展性太差
	\item[3.]dvr:将l3-agent安装在多个计算节点，当vm未分配浮动ip时，访问外网仍旧需要走网络节点。网络节点可以通过传统的HA方案实现。
	遗憾的是，juno不同时支持HA和dvr
\end{itemize}
