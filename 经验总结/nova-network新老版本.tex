\section{nova-network新老版本}

\subsection{nova-network老版本}
\begin{itemize}
	\item[1.]计算节点有两个接口一个内网网口，一个外网网口
	\item[2.]内网之间同一个主机的两个虚拟机，直接通过linux-bridge通信
	\item[3.]内网之间不同主机的两个虚拟机，先通过自己的linux-bridge使用内网口到物理交换机再使用对方的内网口到对方的linux-bridge通信
	\item[4.]访问外网，虚拟机需绑定浮动ip，走外网口。
	\item[5.]linux-bridge内网不同网段可以起到路由的作用，虚拟网络类型包括flat和vlan
\end{itemize}

\subsection{nova-network新版本}
\begin{itemize}
	\item[1.]计算节点只有一个网口，内网和外网的访问都是走这个网口
	\item[2.]虚拟机不需要绑定浮动ip也可以访问外网
	\item[3.]linux-bridge不再扮演内网路由等功能，全部交给物理交换机
	\item[4.]网络类型只有vlan
\end{itemize}
